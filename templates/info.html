AI-Powered Jaundice Detection

Using Computer Vision and Deep Learning to Detect Jaundice from Eye Images

About the Project

This AI-powered jaundice detection tool is designed to help users screen for potential signs of jaundice using advanced image processing and machine learning. By analyzing the sclera (white part of the eyes) for discoloration patterns, the system provides an efficient and accessible health pre-screening solution.

The goal is to reduce barriers to early diagnosis by enabling users to upload an eye photo or enter medical data, which the system then evaluates. The tool is particularly useful in remote or underserved areas, or as a first-step health checker before visiting a healthcare provider. It supports both visual and numerical input for more comprehensive results.

How It Works

The user starts by uploading an image of the eyes or by entering data from medical test reports. The system preprocesses the image and extracts color and texture features, especially focusing on yellowish discoloration in the sclera.

In parallel, any numerical input (like bilirubin levels) is also analyzed. The combined inputs are passed to a trained deep learning model, which evaluates the likelihood of jaundice being present. The result is then displayed clearly, along with confidence scores, enabling users to take timely medical action if needed.

Results

The system demonstrates impressive accuracy in predicting jaundice based on the provided eye image or medical data. With over 90% accuracy achieved on a custom dataset, this tool is a reliable assistant for early detection.

After processing the eye image or medical report, the system outputs the likelihood of jaundice, helping users determine the need for further medical consultation. The result is displayed with a confidence score to show how certain the model is in its prediction, providing clear guidance for next steps.
